# 特点

- **SukiSU**: SukiSU 是安卓 GKI 设备的 root 解决方案，它工作于内核模式，可直接在内核空间为用户空间应用程序授予 root 权限。
- **SUSFS**: 用于 SukiSU 的根隐藏内核补丁和用户空间模块插件。
